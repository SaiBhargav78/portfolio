My Works

Diet Tracker

This project highlight the facts about living a healthy life. Our website allows us to plan a Healthy diet plan. It gives a brief knowledge about variety types of foods and gives us a proper idea on how much quantity of food to take and what kind of food we need to take. This project is made using HTMl, CSS, Javascript and MongoDB(to store our details).

Real-Time Text Translation

This project offers a significant advancement in the realm of text translation and emotion detection compared to existing applications. Unlike traditional text translation apps, which often provide basic translation services, our project integrates real-time screen capture capabilities, enabling users to translate text directly from any on-screen content.

Automating-Flappy-Bird-with-Python

This project goal is to automate a classic game named Flappy Bird. The concept of Deep learning, convolution and max pooling is used to achieve this goal.We used Python v3.9 and tensorflow v2.7. This project follows the description of the Deep Q Learning algorithm described in Playing Atari with Deep Reinforcement Learning and shows that this learning algorithm can be further generalized to the notorious Flappy Bird.
